\section{Functions}
To define a function you use the notation in the following sage cell:
\begin{onlineOnly}
\begin{sageCell}
f(x)=x^5+3*x+4
\end{sageCell}
\end{onlineOnly}
\begin{question}
What output did you get from evaluating the sage cell?

\end{multipleChoice}
